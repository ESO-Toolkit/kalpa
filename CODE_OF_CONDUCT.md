# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

If you experience or witness unacceptable behavior, please report it by opening a [GitHub issue](https://github.com/ESO-Toolkit/kalpa/issues) or contacting the maintainers directly.

We are committed to providing a welcoming and inclusive experience for everyone.
